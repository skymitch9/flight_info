# Cyberpunk Images

Drop your Cyberpunk Edgerunners / 2077 images here.

## Expected files (used in the UI):

| Filename | Where it's used | Suggested image |
|----------|----------------|-----------------|
| `hero-bg.jpg` | Full-page background (subtle, behind UI) | Night City skyline or Edgerunners group shot |
| `card-david.jpg` | Trip list card accent | David Martinez |
| `card-lucy.jpg` | Trip detail page accent | Lucy |
| `card-rebecca.jpg` | Alternate card accent | Rebecca |
| `logo-arasaka.png` | Could use as a decorative element | Arasaka logo or Night City corp logo |
| `scene-city.jpg` | Empty state background | Night City street scene |

## Tips:
- Use high-res images (1920x1080 or larger for backgrounds)
- JPG for photos, PNG for logos/transparent images
- The UI applies opacity overlays so even bright images work fine
